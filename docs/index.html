ReChain

ReChain is a modified blockchain design that uses signature blocks to enforce blockchain integrity rather than an expensive proof-of-work calculation. There is still a fixed proof-of-work difficulty in order to stop many transactions from being executed at once.

Documentation

Generated documentation is available here. Currently, there are no written tutorials or guides because the project isn't that far along yet.
